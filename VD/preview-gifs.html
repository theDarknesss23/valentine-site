Перегляд гіфок

Натисни «Видалити» під гіфкою, яку хочеш прибрати з сторінки. Внизу з’явиться новий список — скопіюй його та заміни масив CUTE_GIFS у index.html (або напиши в чат номери для видалення, і я оновлю файл).

Кроки: відкрий index.html → знайди const CUTE_GIFS = [ ... ]; → заміни вміст масиву на текст з поля вище.
